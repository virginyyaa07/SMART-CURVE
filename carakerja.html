Cara Kerja SMART CURVE

Sistem peringatan dini tikungan rawan kecelakaan berbasis IoT

🚗 1. Kendaraan Mendekat

Kendaraan memasuki area deteksi sebelum tikungan.

📡 2. Sensor HC-SR04 Mendeteksi

Sensor ultrasonik mendeteksi keberadaan kendaraan dan mengukur jaraknya.

🧠 3. ESP32 Memproses Data

Data dari sensor dikirim ke ESP32 untuk dianalisis.

🚦 4. Lampu Peringatan Menyala

LED merah menyala sebagai peringatan kepada pengendara dari arah berlawanan.

🔊 5. Buzzer Aktif

Buzzer memberikan peringatan suara tambahan.

🌐 6. Data Tampil di Dashboard

Status sistem dapat dipantau melalui website SMART CURVE.
